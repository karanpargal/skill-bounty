# Solana Ecosystem Reference

Verified reference for Solana ecosystem programs, platforms, and integrations. Grant and strategy modules must only reference programs listed here.

> **Important:** Program details change frequently. Verify current requirements on official sites before applying. Never invent programs not listed here.

## Grant & Funding Programs

### Solana Foundation Grants

| Field            | Details                                                                                   |
| ---------------- | ----------------------------------------------------------------------------------------- |
| **Program**      | Solana Foundation Grants                                                                  |
| **URL**          | https://solana.org/grants                                                                 |
| **Amount**       | $10K–$250K (varies by scope)                                                              |
| **Best for**     | Open-source tooling, infrastructure, public goods, research                               |
| **Requirements** | Working prototype or strong technical proposal, clear milestones                          |
| **Timeline**     | Rolling applications, 4–8 week review                                                     |
| **Tips**         | Emphasize ecosystem benefit, open-source commitment, Solana-specific technical advantages |

### Colosseum

| Field            | Details                                                              |
| ---------------- | -------------------------------------------------------------------- |
| **Program**      | Colosseum Hackathon + Accelerator                                    |
| **URL**          | https://www.colosseum.org                                            |
| **Amount**       | Hackathon prizes + $250K accelerator investment                      |

## Ecosystem Participation Paths

### For Early-Stage (`idea` → `building`)

1. Join a local Superteam chapter
2. Participate in Colosseum hackathon
3. Build MVP on devnet
4. Engage on Solana Discord/forums

### For Growth-Stage (`devnet` → `mainnet-beta`)

1. Apply for Solana Foundation grant
2. Apply for Superteam grant
3. Integrate Helius + Jupiter (if applicable)
4. Submit to Solana Radar directory

### For Launched (`launched` → `scaling`)

1. Apply for follow-on grants
2. Explore Colosseum accelerator
3. Set up bug bounty on Immunefi
4. Publish analytics on Dune

## What NOT to Reference

Do not invent or reference:

- Specific funding amounts guaranteed by any program
- Application deadlines not verified on official sites
- Partnership opportunities not confirmed by the user
- Token launchpads or IDO platforms (out of scope for MVP)
